This plugin populate the trending/suggested modules from the custom modules of bizlms.
It helps the end user to get and info about modules Like classrooms, learningplans, Programs and certifications including the default moodle course according to the settings defined.
The modules are seggregated into three types
1) Trending Modules : Here the modules that are higly rated in the LMS are populated to the user where he gets and info about the module and can enrol/request to the module.

2) Suggested Modules : Here the modules that are suggested to the user on his preferences/enrollments in the LMS are tracked and populated for enrol/request to the module by the user.

3) Both:Here both the suggestions and ratings are considered in population of the modules to the user.
